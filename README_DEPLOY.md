# Biomech Wild Studio — публикация по ссылке

Эта папка `site/` — готовый статический сайт.

## Что внутри

- `index.html` — красивый интерфейс.
- `test/` — отдельная тестовая рубрика TEST FORMATS.
- `data/cards.json` — данные карточек.
- `data/test_characters.json` — данные TEST-персонажей.

## Как обновлять

После любых изменений карточек запускай из папки проекта:

```bash
python3 tools/build_site.py
python3 tools/build_test_site.py
```

## Как получить отдельную ссылку

Самый простой путь:

1. Создать приватный или публичный GitHub-репозиторий.
2. Загрузить туда содержимое папки `site/`.
3. Включить GitHub Pages.

Альтернатива быстрее:

1. Зарегистрироваться в Netlify или Vercel.
2. Создать новый Static Site.
3. Загрузить папку `site/`.

Важно: сами большие mp4 лучше хранить отдельно. Интерфейс управляет промптами,
подписями и статусами; видео можно держать локально, в Google Drive, YouTube
или отдельном облаке.
